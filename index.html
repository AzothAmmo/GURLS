GURLS: a Toolbox for Large Scale Multiclass Learning

GURLS (Grand Unified Regularized Least Squares) allows for an easy design and efficient execution of machine learning experiments in Matlab by relying on peculiar algorithmic properties of regularized least-squares optimization approach. The toolbox can handle supervised binary classification as well as regression problems. However, it has been specifically tailored to multi-category classification and its extensions like multi-label classification where one input can belong to more than one category. The project includes a native C++ version of the toolbox (named GURLS++). Although the current version is still preliminary and includes only a first instance of optimization strategy, it can be downloaded, easily compiled and used. Feedbacks and suggestions are welcome.

Dependencies

GURLS is a pure Matlab toolbox and hasn't any specific dependecy other that fairly standard Matlab toolboxes. GURLS++ requires: - any implementation of BLAS/LAPACK - BOOST serialization
